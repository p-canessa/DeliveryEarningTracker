Menú Lateral

El menú lateral de Delivery Earning Tracker te permite navegar entre las diferentes secciones de la aplicación. Para abrirlo, toca el ícono del menú en la esquina superior izquierda de la pantalla principal.

Elementos del Menú

  • Exportar Datos: Te permite exportar datos en formato CSV. Para instrucciones, consulta la sección Exportación de Datos.
  • Importar Nómina: Te permite cargar tus nóminas de pago en formato PDF. Para instrucciones, consulta la sección Importación de Nómina.
  • Ver Nómina Mensual: Te permite visualizar tus nóminas de pago. Para instrucciones, consulta la sección Visualización de Nómina.
  • Resumen Trimestral: Muestra estadísticas de tus ganancias trimestralmente. Para detalles, consulta la sección Resúmenes.
  • Resumen Anual: Muestra estadísticas de tus ganancias anualmente. Para detalles, consulta la sección Resúmenes.
  • Reconciliación de Nóminas: Te permite ver las diferencias entre los datos de nóminas cargados y los pedidos ingresados en la base de datos y reconciliarlos. Para instrucciones, consulta la sección Reconciliación de Nóminas.
  • Ajustes: Modifica la configuración de tu aplicación. Para instrucciones, consulta la sección Ajustes.
