Configuración

La pantalla de configuración te permite personalizar la aplicación según tus preferencias. Puedes acceder a esta vista seleccionando la opción "Configuración" desde el menú lateral.

Opciones Disponibles

La pantalla de configuración incluye las siguientes opciones:

  • Símbolo de Moneda: Te permite seleccionar la moneda a usar en la app (por ejemplo, €, $, £). El valor predeterminado es €.
  • Modo Noche: Un interruptor para activar o desactivar el tema oscuro de la app. Si está desactivado (valor predeterminado), la app usa el tema claro.
  • Copia de Seguridad de Datos: Crea una copia de la base de datos de la app en una carpeta de tu elección en tu dispositivo. Útil para guardar tus datos en caso de necesidad.
  • Restaurar Datos: Te permite restaurar la base de datos de la app desde una copia de seguridad creada previamente. Asegúrate de tener un archivo de copia válido.
  • Restablecer Configuración: Restablece todas las configuraciones de la app a sus valores predeterminados. Nota: esta acción no afecta los datos guardados (por ejemplo, pedidos o nóminas).
  • Eliminar Anuncios: Te permite suscribirte para eliminar los anuncios publicitarios y desbloquear todas las funciones de la app.
  • Compartir Estadísticas Anónimas: Un interruptor para permitir el envío de datos agregados anónimos. Al activar esta opción (desactivada por defecto), ayudas a los desarrolladores a mejorar la app.

Cómo Usar la Configuración

  • Accede a la pantalla de configuración desde el menú lateral.
  • Modifica las opciones según tus preferencias:
    • Selecciona una moneda diferente si no usas el euro.
    • Activa el "Modo Noche" para una experiencia visual más cómoda en condiciones de poca luz.
    • Realiza una copia de seguridad de los datos regularmente para evitar perder información importante.
    • Usa "Restaurar Datos" solo si tienes una copia de seguridad válida y deseas sobrescribir los datos actuales.
  • Si encuentras problemas con la configuración, puedes usar la opción "Restablecer Configuración" para volver a los valores predeterminados.
