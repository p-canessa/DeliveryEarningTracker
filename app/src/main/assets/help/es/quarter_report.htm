Resúmenes Trimestrales

La pantalla de resúmenes trimestrales te permite visualizar las estadísticas de tus ganancias trimestralmente, especialmente útil para trabajadores autónomos con obligaciones de IVA estándar que necesitan pagar el IVA trimestral. Puedes acceder a esta vista seleccionando la opción "Resumen Trimestral" desde el menú lateral.

Detalles de la Pantalla

La pantalla consta de los siguientes elementos principales:

  • Menú Desplegable: En la parte superior, hay un menú desplegable que te permite seleccionar el trimestre deseado (por ejemplo, T1 2025, T4 2024, etc.).
  • Tabla de Datos: Una tabla resume tus ganancias con las siguientes columnas y filas:
    • Columnas: Categoría, Bruto, Neto, IVA.
    • Filas: Pedidos, Ajustes, Propinas, Total.

Cómo Usar el Informe Trimestral

  • Asegúrate de haber importado las nóminas relevantes (ver Importación de Nómina).
  • Selecciona el trimestre deseado desde el menú desplegable para ver los datos correspondientes.
  • Analiza los valores en la tabla para entender tus ganancias brutas, netas y el IVA asociado para cada categoría (pedidos, ajustes, propinas) y el total general.
