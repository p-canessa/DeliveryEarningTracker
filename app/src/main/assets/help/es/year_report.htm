Resumen Anual

La pantalla de resumen anual te permite visualizar las estadísticas de tus ganancias anualmente. Este informe es útil tanto para el pago de impuestos de los autónomos como para monitorear el umbral máximo de 5,000 € para trabajadores ocasionales. Puedes acceder a esta vista seleccionando la opción "Resumen Anual" desde el menú lateral.

Período del Informe

El informe anual se calcula "por caja," cubriendo el período desde diciembre del año anterior hasta noviembre del año seleccionado. Esto se debe a que los pagos se realizan el mes siguiente (por ejemplo, para el año 2025, el informe incluye datos desde diciembre de 2024 hasta noviembre de 2025).

Detalles de la Pantalla

La pantalla consta de los siguientes elementos principales:

  • Menú Desplegable: En la parte superior, hay un menú desplegable que te permite seleccionar el año deseado (por ejemplo, 2024, 2025, etc.).
  • Tabla de Datos: Una tabla resume tus ganancias con las siguientes columnas y filas:
    • Columnas: Categoría, Bruto, Neto, IVA.
    • Filas: Pedidos, Ajustes, Propinas, Total.
  • Información sobre Entregas: Debajo de la tabla, se indica:
    • El número de entregas realizadas durante el año.
    • El número de entregas faltantes para alcanzar el bono (los riders italianos reciben un bono cada 2,000 entregas).

Cómo Usar el Informe Anual

  • Asegúrate de haber ingresado los pedidos e importado las nóminas relevantes (ver Ingreso Manual, Ingreso por OCR y Importación de Nómina).
  • Selecciona el año deseado desde el menú desplegable para ver los datos correspondientes.
  • Analiza los valores en la tabla para entender tus ganancias brutas, netas y el IVA asociado para cada categoría (pedidos, ajustes, propinas) y el total general.
  • Revisa el número de entregas realizadas y las faltantes para el bono, para planificar tu trabajo y alcanzar la meta de 2,000 entregas.
