Visualización de Nómina

Esta pantalla te permite visualizar los datos de una nómina importada previamente. Puedes acceder a esta vista seleccionando la opción "Mostrar Nómina Mensual" desde el menú lateral.

Detalles de la Pantalla

La pantalla muestra los detalles de la nómina para el mes seleccionado, como las ganancias totales, propinas y otra información relevante. Usa los dos botones y para navegar entre las nóminas presentes en la base de datos.

Cómo Acceder a las Nóminas

  • Asegúrate de haber importado una o más nóminas usando la función Importación de Nómina.
  • Selecciona la nómina deseada para ver los datos correspondientes.
