Importación de Nómina

La función de importación de nómina te permite leer los datos de la nómina de fin de mes enviada por Deliveroo. Estos datos pueden utilizarse para generar resúmenes trimestrales y anuales, así como para realizar la reconciliación de los pedidos ingresados.

Cómo Importar una Nómina

  1. Selecciona la función "Importar Nómina" desde el menú lateral.
  2. Selecciona un documento con el nombre `invoice_*.pdf` desde tu dispositivo usando el botón "IMPORTAR NÓMINA" (esto podría requerir ver un anuncio).
  3. La aplicación extraerá automáticamente los datos del archivo PDF. Puedes importar varias nóminas para el mismo mes, siempre que tengan numeraciones diferentes.
  4. Si la nómina no está ya en la base de datos, se añadirá y se mostrará un mensaje de éxito. De lo contrario, se mostrará un mensaje de error.
  5. En este punto, la nómina puede visualizarse en la sección Mostrar Nómina Mensual.
