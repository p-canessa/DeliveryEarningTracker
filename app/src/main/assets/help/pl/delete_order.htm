Usuń zamówienie

Możesz usunąć zamówienie z ekranu głównego, jeśli nie jest już potrzebne lub zostało wprowadzone przez pomyłkę.

Jak usunąć zamówienie

  1. Na ekranie głównym znajdź zamówienie, które chcesz usunąć.
  2. Przytrzymaj zamówienie (długie naciśnięcie), aby otworzyć menu kontekstowe.
  3. Wybierz opcję "Usuń" z menu.
  4. Potwierdź usunięcie w oknie dialogowym, które się pojawi, lub anuluj, jeśli zmienisz zdanie.

Uwaga: Usunięcie zamówienia jest nieodwracalne. Jeśli chcesz edytować zamówienie zamiast je usuwać, zobacz sekcję Edytuj zamówienie.
