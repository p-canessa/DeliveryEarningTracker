Експорт Даних

Екран експорту даних дозволяє експортувати зареєстровані замовлення у файл CSV для подальшого аналізу (наприклад, у таблиці). Ви можете отримати доступ до цього перегляду, вибравши опцію "Експорт Даних" у бічному меню.

Деталі Екрану

Екран містить наступні основні елементи:

  • Поля Дат: Два поля для вибору періоду експорту:
    • Дата Початку: Дата початку періоду у форматі `yyyy-MM-dd` (наприклад, 2025-01-01).
    • Дата Закінчення: Дата закінчення періоду у форматі `yyyy-MM-dd` (наприклад, 2025-12-31).
  • Кнопка ЕКСПОРТУВАТИ CSV: Кнопка для запуску експорту замовлень у файл CSV. Якщо поля дат заповнені, будуть експортовані лише замовлення за вказаний період. Якщо поля порожні, будуть експортовані всі замовлення.

Як Експортувати Дані

  • Переконайтесь, що ви ввели замовлення, які бажаєте експортувати (див. Ручне Введення та Введення через OCR).
  • Введіть "Дату Початку" та "Дату Закінчення" у відповідні поля, щоб відфільтрувати замовлення для експорту. Залиште поля порожніми, щоб експортувати всі замовлення.
  • Торкніться кнопки "ЕКСПОРТУВАТИ CSV", щоб згенерувати файл CSV.
  • Файл CSV буде збережено на вашому пристрої і може бути відкритий у програмі для роботи з таблицями для подальшого аналізу.
