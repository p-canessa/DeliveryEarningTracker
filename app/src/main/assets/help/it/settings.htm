Impostazioni

La schermata delle impostazioni ti permette di personalizzare l'app in base alle tue preferenze. Puoi accedere a questa vista selezionando l'opzione "Impostazioni" dal menu laterale.

Opzioni Disponibili

La schermata delle impostazioni include le seguenti opzioni:

  • Simbolo Valuta: Ti permette di selezionare la valuta da usare nell'app (es. €, $, £). Il valore predefinito è €.
  • Modalità Notte: Un interruttore per attivare o disattivare il tema scuro dell'app. Se disattivato (valore predefinito), l'app utilizza il tema chiaro.
  • Backup Dati: Crea una copia del database dell'app in una cartella scelta sul tuo dispositivo. Utile per salvare i tuoi dati in caso di necessità.
  • Ripristina Dati: Ti permette di ripristinare il database dell'app da un backup precedentemente creato. Assicurati di avere un file di backup valido.
  • Reset Impostazioni: Ripristina tutte le impostazioni dell'app ai valori predefiniti. Nota: questa azione non influisce sui dati salvati (es. ordini o statini).
  • Rimuovi gli Annunci: Ti permette di abbonarti per rimuovere gli annunci pubblicitari e sbloccare tutte le funzionalità dell'app.
  • Condividi Statistiche Anonime: Un interruttore per consentire l'invio di dati aggregati anonimizzati. Attivando questa opzione (disattivata per default), aiuti gli sviluppatori a migliorare l'app.

Come utilizzare le impostazioni

  • Accedi alla schermata delle impostazioni dal menu laterale.
  • Modifica le opzioni in base alle tue preferenze:
    • Seleziona una valuta diversa se non utilizzi l'euro.
    • Attiva la "Modalità Notte" per un'esperienza visiva più confortevole in condizioni di scarsa illuminazione.
    • Esegui un backup dei dati regolarmente per evitare la perdita di informazioni importanti.
    • Usa "Ripristina Dati" solo se hai un backup valido e desideri sovrascrivere i dati attuali.
  • Se riscontri problemi con le impostazioni, puoi utilizzare l'opzione "Reset Impostazioni" per tornare ai valori predefiniti.
