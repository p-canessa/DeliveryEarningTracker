Inserimento tramite OCR

La funzione di inserimento tramite OCR ti permette di aggiungere un ordine scansionando uno screenshot dall'app rider, direttamente dalla schermata principale.

Come inserire un ordine tramite OCR

  1. Dalla schermata principale, tocca il pulsante di destra per la scansione OCR.
  2. Se devi inserire un ordine per una data diversa da quella dello screenshot puoi sovrascrivere la data tramite il campo di testo e lo switcch usa data
  3. Seleziona uno screenshot dal tuo dispositivo tramite il tasto SELEZIONA FILE (può richiedre la visione di un ad).
  4. L'app estrarrà automaticamente i dettagli dell'ordine (es. numero di consegne, totale guadagnato).
  5. Se nec essario inserisci l'importo della Mancia in Contanti.
  6. Verifica i dati estratti, se necessario modificali e conferma l'inserimento.

Risoluzione dei problemi

Se l'OCR non riconosce correttamente i dati, considera quanto segue:

  • Assicurati che lo screenshot sia chiaro e leggibile.
  • Verifica che il formato dello screenshot sia compatibile con l'app rider.
  • Se il problema persiste, puoi inserire l'ordine manualmente. Consulta la sezione Inserimento Manuale.
