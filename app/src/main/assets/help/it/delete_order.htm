Elimina Ordine

Puoi eliminare un ordine dalla schermata principale se non è più necessario o se è stato inserito per errore.

Come eliminare un ordine

  1. Dalla schermata principale, individua l'ordine che desideri eliminare.
  2. Tieni premuto sull'ordine (tocco prolungato) per aprire il menu contestuale.
  3. Seleziona l'opzione "Elimina" dal menu.
  4. Conferma l'eliminazione nella finestra di dialogo che appare, oppure annulla se cambi idea.

Nota: l'eliminazione di un ordine è irreversibile. Se desideri modificare un ordine invece di eliminarlo, consulta la sezione Modifica Ordine.
