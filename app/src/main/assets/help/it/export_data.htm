Esportazione Dati

La schermata di esportazione dati ti permette di esportare gli ordini registrati in un file CSV per ulteriori analisi (es. con un foglio di calcolo). Puoi accedere a questa vista selezionando l'opzione "Esporta Dati" dal menu laterale.

Dettagli della Schermata

La schermata include i seguenti elementi principali:

  • Campi per le date: Due campi per selezionare il periodo di esportazione:
    • Data Inizio: La data di inizio del periodo, nel formato `yyyy-MM-dd` (es. 2025-01-01).
    • Data Fine: La data di fine del periodo, nel formato `yyyy-MM-dd` (es. 2025-12-31).
  • Pulsante ESPORTA CSV: Un pulsante per avviare l'esportazione degli ordini in un file CSV. Se i campi delle date sono compilati, verranno esportati solo gli ordini compresi nel periodo specificato. Se i campi sono vuoti, verranno esportati tutti gli ordini.

Come esportare i dati

  • Assicurati di aver inserito gli ordini che desideri esportare (vedi Inserimento Manuale e Inserimento tramite OCR).
  • Inserisci la "Data Inizio" e la "Data Fine" nei rispettivi campi per filtrare gli ordini da esportare. Lascia i campi vuoti per esportare tutti gli ordini.
  • Tocca il pulsante "ESPORTA CSV" per generare il file CSV.
  • Il file CSV verrà salvato sul tuo dispositivo e potrà essere aperto con un'applicazione di fogli di calcolo per ulteriori analisi.
