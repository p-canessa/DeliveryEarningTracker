Manual Entry

The manual entry function allows you to add an order directly from the main screen by entering details such as the number of deliveries, total earned, and time spent.

How to Manually Insert an Order

  1. From the main screen, tap the "NEW ORDER" button.
  2. Enter the order details:
    • Date: The date the order was completed, automatically set to todayâ€™s date.
    • Base Pay: The value of the "Orders Delivered" line in the Deliveroo Rider app.
    • Cash Collected: The value of the corresponding line, if present.
    • Extra Pay: The value of any extra pay, if present.
    • Tip: The value of the tip received in the app, if present.
    • Cash Tip: The value of the tip received in cash.
    • Number of Orders: The number of orders delivered in the same batch.
    • Time Spent: The time taken to deliver the orders, which can be expressed in minutes or as a time range in hours and minutes as shown in the Rider app (hh:mm-hh:mm).
  3. Confirm the entry to save the order.

How to Edit an Order

To edit an existing order, follow these steps:

  1. From the main screen, locate the order you want to edit.
  2. Tap the order to open the editing screen (the same screen used for manual entry).
  3. Update the details, such as the number of deliveries, base pay, time spent, or tips.
  4. Confirm the changes to save the updated order.

If you prefer to insert an order by scanning a screenshot, see the OCR Entry section. To delete an order, see the Delete Order section.
