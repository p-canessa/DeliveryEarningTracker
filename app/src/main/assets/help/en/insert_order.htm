OCR Entry

The OCR entry function allows you to add an order by scanning a screenshot from the rider app, directly from the main screen.

How to Insert an Order via OCR

  1. From the main screen, tap the right button for OCR scanning.
  2. If you need to enter an order for a date different from the screenshot, you can override the date using the text field and the "use date" switch.
  3. Select a screenshot from your device using the "SELECT FILE" button (this may require watching an ad).
  4. The app will automatically extract the order details (e.g., number of deliveries, total earned).
  5. If necessary, enter the cash tip amount.
  6. Verify the extracted data, edit it if needed, and confirm the entry.

Troubleshooting

If the OCR does not correctly recognize the data, consider the following:

  • Ensure the screenshot is clear and legible.
  • Check that the screenshot format is compatible with the rider app.
  • If the issue persists, you can enter the order manually. See the Manual Entry section.
