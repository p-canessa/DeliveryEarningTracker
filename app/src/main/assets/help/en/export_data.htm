Data Export

The data export screen allows you to export recorded orders to a CSV file for further analysis (e.g., with a spreadsheet). You can access this view by selecting the "Export Data" option from the side menu.

Screen Details

The screen includes the following main elements:

  • Date Fields: Two fields to select the export period:
    • Start Date: The start date of the period, in the format `yyyy-MM-dd` (e.g., 2025-01-01).
    • End Date: The end date of the period, in the format `yyyy-MM-dd` (e.g., 2025-12-31).
  • EXPORT CSV Button: A button to start exporting the orders to a CSV file. If the date fields are filled, only orders within the specified period will be exported. If the fields are empty, all orders will be exported.

How to Export Data

  • Ensure you have entered the orders you wish to export (see Manual Entry and OCR Entry).
  • Enter the "Start Date" and "End Date" in the respective fields to filter the orders to export. Leave the fields empty to export all orders.
  • Tap the "EXPORT CSV" button to generate the CSV file.
  • The CSV file will be saved on your device and can be opened with a spreadsheet application for further analysis.
