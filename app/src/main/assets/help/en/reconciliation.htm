Reconciliation

This screen allows you to compare the daily order data extracted from a paystub with those entered manually or via OCR. You can access this view by selecting the "Reconciliation" option from the side menu.

Screen Details

The screen includes the following main elements:

  • Month Navigation: Use the two buttons and to navigate between months and view the corresponding reconciliation data.
  • Comparison Table: The table displays daily order data (number and amount) extracted from the paystub alongside those entered manually or via OCR.
    • If the values differ, the text in the table is displayed in red.
    • Clicking on a row with differing values takes you to the list of orders for that day (see Order Viewing).
  • Cash Collected and Tips: Below the order table, two rows show the totals for "Cash Collected" and "Tips":
    • If the values differ from those in the paystub, the text is displayed in red.
    • Clicking on a differing row takes you to the list of orders with cash collected and/or tips other than 0.

How to Use Reconciliation

  • Ensure you have imported one or more paystubs using the Paystub Import function.
  • Enter orders manually or via OCR (see Manual Entry and OCR Entry).
  • Select the desired month using the navigation buttons to view the reconciliation data.
  • Check the table to identify any discrepancies (in red) and click on the rows to view order details and correct errors.
