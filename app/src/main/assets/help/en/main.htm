Delivery Earning Tracker

Delivery Earning Tracker is a tool for monitoring and evaluating earnings generated through home delivery platforms.

Main Features

  • Earnings Tracking, including any tips received directly from customers. You can view your earnings for each order and keep track of tips.
  • Hourly Wage Calculation for both individual orders and daily, weekly, and monthly totals. For example, find out how much you earn on average per hour during a busy week.
  • Paystub Storage to view payment details and reconcile any entry errors. Import your paystubs in PDF format and access the details at any time. For more information, see the Paystub Import section.
  • Quarterly and Annual Statistics useful for tax management (VAT and income payments). View your total earnings by quarter or year for better tax planning. For details, see the Monthly Summaries and Annual Summaries sections.

Main Screen

The main screen of Delivery Earning Tracker allows you to quickly access all the app's features. Below are the main elements of the interface:

Menu and Navigation

  • Side Menu: Tap the icon in the top left corner to open the menu and navigate between sections (e.g., "Paystub Import", "Quarterly Summary"). For a detailed description of the menu items, see the Side Menu section.

Summary and Filters

  • Date Selector: allows you to select a specific date or navigate forward and backward between dates. You can also choose the view by day, week, or month.
  • Period Summary Grid: displays the number of deliveries made, total earnings (including cash received), base pay, cash collected, any extras, total and cash tips, time spent, and average hourly earnings.

Order Entry and Display

  • Order Entry Buttons: the left button ("NEW ORDER") allows manual order entry (see Manual Entry), while the right button enables entry via scanning a screenshot from the rider app (see OCR Entry).
  • Entered Order: shows the order details, emphasizing the number of deliveries, total earnings, time spent, and hourly wage. The icon is displayed if the order received a tip.
  • Edit and Delete Orders: You can edit an order with a tap (see Edit Order) or delete it with a long press (see Delete Order).
