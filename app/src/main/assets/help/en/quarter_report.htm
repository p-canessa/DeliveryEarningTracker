Quarterly Summaries

The quarterly summaries screen allows you to view your earnings statistics on a quarterly basis, particularly useful for self-employed workers with standard VAT obligations who need to pay quarterly VAT. You can access this view by selecting the "Quarterly Summary" option from the side menu.

Screen Details

The screen consists of the following main elements:

  • Dropdown Menu: At the top, there is a dropdown menu that allows you to select the desired quarter (e.g., Q1 2025, Q4 2024, etc.).
  • Data Table: A table summarizes your earnings with the following columns and rows:
    • Columns: Category, Gross, Net, VAT.
    • Rows: Orders, Adjustments, Tips, Total.

How to Use the Quarterly Report

  • Ensure you have imported the relevant paystubs (see Paystub Import).
  • Select the desired quarter from the dropdown menu to view the corresponding data.
  • Analyze the values in the table to understand your gross earnings, net earnings, and associated VAT for each category (orders, adjustments, tips) and the overall total.
