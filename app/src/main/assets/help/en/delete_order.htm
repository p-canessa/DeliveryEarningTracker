Delete Order

You can delete an order from the main screen if it is no longer needed or was entered by mistake.

How to delete an order

  1. From the main screen, locate the order you want to delete.
  2. Long press on the order (hold the touch) to open the context menu.
  3. Select the "Delete" option from the menu.
  4. Confirm the deletion in the dialog box that appears, or cancel if you change your mind.

Note: Deleting an order is irreversible. If you want to edit an order instead of deleting it, see the Edit Order section.
