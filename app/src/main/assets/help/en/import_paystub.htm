Paystub Import

The paystub import function allows you to read the data from the end-of-month paystub sent by Deliveroo. This data can be used to generate quarterly and annual summaries, as well as to perform reconciliation of entered orders.

How to Import a Paystub

  1. Select the "Import Paystub" function from the side menu.
  2. Select a document named `invoice_*.pdf` from your device using the "IMPORT PAYSTUB" button (this may require watching an ad).
  3. The app will automatically extract the data from the PDF file. You can import multiple paystubs for the same month, provided they have different numbering.
  4. If the paystub is not already in the database, it will be added, and a success message will be displayed. Otherwise, an error message will be shown.
  5. At this point, the paystub can be viewed in the Show Monthly Statement section.
