Paystub Viewing

This screen allows you to view the data of a previously imported paystub. You can access this view by selecting the "Show Monthly Statement" option from the side menu.

Screen Details

The screen displays the paystub details for the selected month, such as total earnings, tips, and other relevant information. Use the two buttons and to navigate between paystubs present in the database.

How to Access Paystubs

  • Ensure you have imported one or more paystubs using the Paystub Import function.
  • Select the desired paystub to view the corresponding data.
