Settings

The settings screen allows you to customize the app according to your preferences. You can access this view by selecting the "Settings" option from the side menu.

Available Options

The settings screen includes the following options:

  • Currency Symbol: Allows you to select the currency to use in the app (e.g., €, $, £). The default value is €.
  • Night Mode: A switch to enable or disable the app's dark theme. If disabled (default value), the app uses the light theme.
  • Data Backup: Creates a copy of the app’s database in a folder of your choice on your device. Useful for saving your data if needed.
  • Restore Data: Allows you to restore the app’s database from a previously created backup. Ensure you have a valid backup file.
  • Reset Settings: Resets all app settings to their default values. Note: This action does not affect saved data (e.g., orders or paystubs).
  • Remove Ads: Allows you to subscribe to remove advertisements and unlock all app features.
  • Share Anonymous Statistics: A switch to allow sending anonymized aggregated data. Enabling this option (disabled by default) helps developers improve the app.

How to Use Settings

  • Access the settings screen from the side menu.
  • Modify the options according to your preferences:
    • Select a different currency if you don’t use the euro.
    • Enable "Night Mode" for a more comfortable visual experience in low-light conditions.
    • Perform a data backup regularly to avoid losing important information.
    • Use "Restore Data" only if you have a valid backup and want to overwrite current data.
  • If you encounter issues with the settings, you can use the "Reset Settings" option to return to default values.
