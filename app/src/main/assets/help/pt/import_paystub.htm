Importação de Contracheque

A função de importação de contracheque permite que você leia os dados do contracheque de fim de mês enviado pela Deliveroo. Esses dados podem ser usados para gerar resumos trimestrais e anuais, bem como para realizar a reconciliação dos pedidos inseridos.

Como Importar um Contracheque

  1. Selecione a função "Importar Contracheque" no menu lateral.
  2. Selecione um documento chamado `invoice_*.pdf` do seu dispositivo usando o botão "IMPORTAR CONTRACHEQUE" (pode ser necessário assistir a um anúncio).
  3. O aplicativo extrairá automaticamente os dados do arquivo PDF. Você pode importar vários contracheques para o mesmo mês, desde que tenham numerações diferentes.
  4. Se o contracheque ainda não estiver no banco de dados, ele será adicionado e uma mensagem de sucesso será exibida. Caso contrário, uma mensagem de erro será mostrada.
  5. Neste ponto, o contracheque pode ser visualizado na seção Mostrar Contracheque Mensal.
