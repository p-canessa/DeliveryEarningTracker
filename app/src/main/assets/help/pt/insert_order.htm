Entrada por OCR

A função de entrada por OCR permite que você adicione um pedido escaneando uma captura de tela do aplicativo do rider, diretamente da tela principal.

Como Inserir um Pedido por OCR

  1. A partir da tela principal, toque no botão à direita para a varredura OCR.
  2. Se precisar inserir um pedido para uma data diferente da captura de tela, você pode substituir a data usando o campo de texto e o interruptor "usar data".
  3. Selecione uma captura de tela do seu dispositivo usando o botão "SELECIONAR ARQUIVO" (pode ser necessário assistir a um anúncio).
  4. O aplicativo extrairá automaticamente os detalhes do pedido (por exemplo, número de entregas, total ganho).
  5. Se necessário, insira o valor da gorjeta em dinheiro.
  6. Verifique os dados extraídos, edite-os se necessário e confirme a entrada.

Resolução de Problemas

Se o OCR não reconhecer corretamente os dados, considere o seguinte:

  • Certifique-se de que a captura de tela esteja clara e legível.
  • Verifique se o formato da captura de tela é compatível com o aplicativo do rider.
  • Se o problema persistir, você pode inserir o pedido manualmente. Veja a seção Entrada Manual.
