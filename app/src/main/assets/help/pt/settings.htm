Configurações

A tela de configurações permite personalizar o aplicativo de acordo com suas preferências. Você pode acessar esta visualização selecionando a opção "Configurações" no menu lateral.

Opções Disponíveis

A tela de configurações inclui as seguintes opções:

  • Símbolo da Moeda: Permite selecionar a moeda a ser usada no aplicativo (por exemplo, €, $, £). O valor padrão é €.
  • Modo Noturno: Um interruptor para ativar ou desativar o tema escuro do aplicativo. Se desativado (valor padrão), o aplicativo usa o tema claro.
  • Backup de Dados: Cria uma cópia do banco de dados do aplicativo em uma pasta de sua escolha no dispositivo. Útil para salvar seus dados em caso de necessidade.
  • Restaurar Dados: Permite restaurar o banco de dados do aplicativo a partir de um backup criado anteriormente. Certifique-se de ter um arquivo de backup válido.
  • Redefinir Configurações: Redefine todas as configurações do aplicativo para os valores padrão. Nota: Esta ação não afeta os dados salvos (por exemplo, pedidos ou contracheques).
  • Remover Anúncios: Permite assinar para remover anúncios e desbloquear todas as funcionalidades do aplicativo.
  • Compartilhar Estatísticas Anônimas: Um interruptor para permitir o envio de dados agregados anônimos. Ao ativar esta opção (desativada por padrão), você ajuda os desenvolvedores a melhorar o aplicativo.

Como Usar as Configurações

  • Acesse a tela de configurações pelo menu lateral.
  • Modifique as opções conforme suas preferências:
    • Selecione uma moeda diferente se não usar o euro.
    • Ative o "Modo Noturno" para uma experiência visual mais confortável em condições de pouca luz.
    • Realize um backup dos dados regularmente para evitar a perda de informações importantes.
    • Use "Restaurar Dados" apenas se tiver um backup válido e quiser substituir os dados atuais.
  • Se encontrar problemas com as configurações, use a opção "Redefinir Configurações" para voltar aos valores padrão.
