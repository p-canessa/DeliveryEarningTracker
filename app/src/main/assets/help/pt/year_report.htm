Resumo Anual

A tela de resumo anual permite visualizar as estatísticas dos seus ganhos anualmente. Este relatório é útil tanto para o pagamento de impostos para freelancers quanto para monitorar o limite máximo de 5.000 € para trabalhadores ocasionais. Você pode acessar esta visualização selecionando a opção "Resumo Anual" no menu lateral.

Período do Relatório

O relatório anual é calculado "em base de caixa," cobrindo o período de dezembro do ano anterior a novembro do ano selecionado. Isso ocorre porque os pagamentos são feitos no mês seguinte (por exemplo, para o ano de 2025, o relatório inclui dados de dezembro de 2024 a novembro de 2025).

Detalhes da Tela

A tela é composta pelos seguintes elementos principais:

  • Menu Suspenso: No topo, há um menu suspenso que permite selecionar o ano desejado (por exemplo, 2024, 2025, etc.).
  • Tabela de Dados: Uma tabela resume seus ganhos com as seguintes colunas e linhas:
    • Colunas: Categoria, Bruto, Líquido, IVA.
    • Linhas: Pedidos, Ajustes, Gorjetas, Total.
  • Informações sobre Entregas: Abaixo da tabela, são indicados:
    • O número de entregas realizadas durante o ano.
    • O número de entregas restantes para alcançar o bônus (os riders italianos recebem um bônus a cada 2.000 entregas).

Como Usar o Relatório Anual

  • Certifique-se de ter inserido os pedidos e importado os contracheques relevantes (veja Entrada Manual, Entrada por OCR e Importação de Contracheque).
  • Selecione o ano desejado no menu suspenso para visualizar os dados correspondentes.
  • Analise os valores na tabela para entender seus ganhos brutos, líquidos e o IVA associado para cada categoria (pedidos, ajustes, gorjetas) e o total geral.
  • Verifique o número de entregas realizadas e as restantes para o bônus, para planejar seu trabalho e atingir a meta de 2.000 entregas.
