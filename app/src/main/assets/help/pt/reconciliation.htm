Reconciliação

Esta tela permite comparar os dados de pedidos diários extraídos de um contracheque com aqueles inseridos manualmente ou via OCR. Você pode acessar esta visualização selecionando a opção "Reconciliação" no menu lateral.

Detalhes da Tela

A tela inclui os seguintes elementos principais:

  • Navegação entre Meses: Use os dois botões e para navegar entre os meses e visualizar os dados de reconciliação correspondentes.
  • Tabela de Comparação: A tabela exibe os dados de pedidos diários (número e valor) extraídos do contracheque ao lado daqueles inseridos manualmente ou via OCR.
    • Se os valores diferirem, o texto na tabela será exibido em vermelho.
    • Ao clicar em uma linha com valores diferentes, você acessa a lista de pedidos daquele dia (veja Visualização de Pedidos).
  • Dinheiro Coletado e Gorjetas: Abaixo da tabela de pedidos, duas linhas mostram os totais de "Dinheiro Coletado" e "Gorjetas":
    • Se os valores diferirem dos do contracheque, o texto será exibido em vermelho.
    • Ao clicar em uma linha diferente, você acessa a lista de pedidos com dinheiro coletado e/ou gorjetas diferentes de 0.

Como Usar a Reconciliação

  • Certifique-se de ter importado um ou mais contracheques usando a função Importação de Contracheque.
  • Insira os pedidos manualmente ou via OCR (veja Entrada Manual e Entrada por OCR).
  • Selecione o mês desejado usando os botões de navegação para visualizar os dados de reconciliação.
  • Verifique a tabela para identificar discrepâncias (em vermelho) e clique nas linhas para ver os detalhes dos pedidos e corrigir erros.
