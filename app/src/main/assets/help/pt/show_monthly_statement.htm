Visualização de Contracheque

Esta tela permite visualizar os dados de um contracheque importado anteriormente. Você pode acessar esta visualização selecionando a opção "Mostrar Contracheque Mensal" no menu lateral.

Detalhes da Tela

A tela exibe os detalhes do contracheque para o mês selecionado, como ganhos totais, gorjetas e outras informações relevantes. Use os dois botões e para navegar entre os contracheques presentes no banco de dados.

Como Acessar os Contracheques

  • Certifique-se de ter importado um ou mais contracheques usando a função Importação de Contracheque.
  • Selecione o contracheque desejado para visualizar os dados correspondentes.
