Menu Lateral

O menu lateral do Delivery Earning Tracker permite navegar entre as diferentes seções do aplicativo. Para abri-lo, toque no ícone de menu no canto superior esquerdo da tela principal.

Itens do Menu

  • Exportar dados: Permite exportar dados em formato CSV. Para instruções, consulte a seção Exportação de Dados.
  • Importar Contracheque: Permite carregar seus contracheques em formato PDF. Para instruções, consulte a seção Importação de Contracheque.
  • Visualizar Contracheque Mensal: Permite visualizar seus contracheques. Para instruções, consulte a seção Visualização de Contracheque.
  • Resumo Trimestral: Exibe estatísticas dos seus ganhos trimestralmente. Para detalhes, veja a seção Resumos.
  • Resumo Anual: Exibe estatísticas dos seus ganhos anualmente. Para detalhes, veja a seção Resumos.
  • Reconciliação de Contracheques: Permite visualizar as diferenças entre os dados dos contracheques carregados e os pedidos inseridos no banco de dados e reconciliá-los. Para instruções, consulte a seção Reconciliação de Contracheques.
  • Configurações: Modifique as configurações do seu aplicativo. Para instruções, consulte a seção Configurações.
