Exportação de Dados

A tela de exportação de dados permite exportar os pedidos registrados para um arquivo CSV para análise adicional (por exemplo, com uma planilha). Você pode acessar esta visualização selecionando a opção "Exportar Dados" no menu lateral.

Detalhes da Tela

A tela inclui os seguintes elementos principais:

  • Campos de Data: Dois campos para selecionar o período de exportação:
    • Data de Início: A data de início do período, no formato `yyyy-MM-dd` (por exemplo, 2025-01-01).
    • Data de Término: A data de término do período, no formato `yyyy-MM-dd` (por exemplo, 2025-12-31).
  • Botão EXPORTAR CSV: Um botão para iniciar a exportação dos pedidos para um arquivo CSV. Se os campos de data estiverem preenchidos, apenas os pedidos dentro do período especificado serão exportados. Se os campos estiverem vazios, todos os pedidos serão exportados.

Como Exportar Dados

  • Certifique-se de ter inserido os pedidos que deseja exportar (veja Entrada Manual e Entrada por OCR).
  • Insira a "Data de Início" e a "Data de Término" nos respectivos campos para filtrar os pedidos a exportar. Deixe os campos vazios para exportar todos os pedidos.
  • Toque no botão "EXPORTAR CSV" para gerar o arquivo CSV.
  • O arquivo CSV será salvo no seu dispositivo e poderá ser aberto com um aplicativo de planilhas para análise adicional.
