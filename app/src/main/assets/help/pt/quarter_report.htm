Resumos Trimestrais

A tela de resumos trimestrais permite visualizar as estatísticas dos seus ganhos trimestralmente, especialmente útil para trabalhadores autônomos com obrigações de IVA padrão que precisam pagar o IVA trimestral. Você pode acessar esta visualização selecionando a opção "Resumo Trimestral" no menu lateral.

Detalhes da Tela

A tela é composta pelos seguintes elementos principais:

  • Menu Suspenso: No topo, há um menu suspenso que permite selecionar o trimestre desejado (por exemplo, T1 2025, T4 2024, etc.).
  • Tabela de Dados: Uma tabela resume seus ganhos com as seguintes colunas e linhas:
    • Colunas: Categoria, Bruto, Líquido, IVA.
    • Linhas: Pedidos, Ajustes, Gorjetas, Total.

Como Usar o Relatório Trimestral

  • Certifique-se de ter importado os contracheques relevantes (veja Importação de Contracheque).
  • Selecione o trimestre desejado no menu suspenso para visualizar os dados correspondentes.
  • Analise os valores na tabela para entender seus ganhos brutos, líquidos e o IVA associado para cada categoria (pedidos, ajustes, gorjetas) e o total geral.
