Supprimer une commande

Vous pouvez supprimer une commande depuis l'écran principal si elle n'est plus nécessaire ou si elle a été saisie par erreur.

Comment supprimer une commande

  1. Depuis l'écran principal, repérez la commande que vous souhaitez supprimer.
  2. Maintenez la commande enfoncée (appui long) pour ouvrir le menu contextuel.
  3. Sélectionnez l'option "Supprimer" dans le menu.
  4. Confirmez la suppression dans la boîte de dialogue qui apparaît, ou annulez si vous changez d'avis.

Note : La suppression d'une commande est irréversible. Si vous souhaitez modifier une commande au lieu de la supprimer, consultez la section Modifier une commande.
