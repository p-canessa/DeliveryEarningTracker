Exportation des Données

L'écran d'exportation des données vous permet d'exporter les commandes enregistrées dans un fichier CSV pour une analyse plus approfondie (par exemple, avec un tableur). Vous pouvez accéder à cette vue en sélectionnant l'option "Exporter les Données" dans le menu latéral.

Détails de l'Écran

L'écran comprend les éléments principaux suivants :

  • Champs de dates : Deux champs pour sélectionner la période d'exportation :
    • Date de Début : La date de début de la période, au format `yyyy-MM-dd` (par exemple, 2025-01-01).
    • Date de Fin : La date de fin de la période, au format `yyyy-MM-dd` (par exemple, 2025-12-31).
  • Bouton EXPORTER CSV : Un bouton pour lancer l'exportation des commandes dans un fichier CSV. Si les champs de dates sont remplis, seules les commandes de la période spécifiée seront exportées. Si les champs sont vides, toutes les commandes seront exportées.

Comment exporter les données

  • Assurez-vous d'avoir saisi les commandes que vous souhaitez exporter (voir Saisie Manuelle et Saisie par OCR).
  • Entrez la "Date de Début" et la "Date de Fin" dans les champs respectifs pour filtrer les commandes à exporter. Laissez les champs vides pour exporter toutes les commandes.
  • Appuyez sur le bouton "EXPORTER CSV" pour générer le fichier CSV.
  • Le fichier CSV sera enregistré sur votre appareil et pourra être ouvert avec une application de tableur pour une analyse plus approfondie.
