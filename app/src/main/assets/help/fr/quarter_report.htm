Résumés Trimestriels

L'écran des résumés trimestriels vous permet de visualiser les statistiques de vos gains sur une base trimestrielle, particulièrement utile pour les travailleurs indépendants soumis à la TVA standard qui doivent payer la TVA trimestrielle. Vous pouvez accéder à cette vue en sélectionnant l'option "Résumé Trimestriel" dans le menu latéral.

Détails de l'Écran

L'écran comprend les éléments principaux suivants :

  • Menu Déroulant: En haut, un menu déroulant vous permet de sélectionner le trimestre souhaité (par exemple, T1 2025, T4 2024, etc.).
  • Tableau de Données: Un tableau résume vos gains avec les colonnes et lignes suivantes :
    • Colonnes: Catégorie, Brut, Net, TVA.
    • Lignes: Commandes, Ajustements, Pourboires, Total.

Comment Utiliser le Rapport Trimestriel

  • Assurez-vous d'avoir importé les relevés pertinents (voir Importation de Relevé).
  • Sélectionnez le trimestre souhaité dans le menu déroulant pour afficher les données correspondantes.
  • Analysez les valeurs dans le tableau pour comprendre vos gains bruts, nets et la TVA associée pour chaque catégorie (commandes, ajustements, pourboires) et le total global.
