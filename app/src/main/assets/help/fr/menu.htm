Menu Latéral

Le menu latéral de Delivery Earning Tracker vous permet de naviguer entre les différentes sections de l'application. Pour l'ouvrir, touchez l'icône du menu en haut à gauche de l'écran principal.

Éléments du Menu

  • Exporter les Données: Vous permet d'exporter les données au format CSV. Pour les instructions, consultez la section Exportation des Données.
  • Importer un Relevé: Vous permet de charger vos relevés de paiement au format PDF. Pour les instructions, consultez la section Importation de Relevé.
  • Voir le Relevé Mensuel: Vous permet de visualiser vos relevés de paiement. Pour les instructions, consultez la section Visualisation de Relevé.
  • Résumé Trimestriel: Affiche les statistiques de vos gains sur une base trimestrielle. Pour plus de détails, voir la section Résumés.
  • Résumé Annuel: Affiche les statistiques de vos gains sur une base annuelle. Pour plus de détails, voir la section Résumés.
  • Réconciliation des Relevés: Vous permet de voir les différences entre les données des relevés chargés et les commandes saisies dans la base de données et de les réconcilier. Pour les instructions, consultez la section Réconciliation des Relevés.
  • Paramètres: Modifiez les paramètres de votre application. Pour les instructions, consultez la section Paramètres.
