Résumé Annuel

L'écran du résumé annuel vous permet de visualiser les statistiques de vos gains sur une base annuelle. Ce rapport est utile à la fois pour le paiement des impôts pour les indépendants et pour surveiller le seuil maximum de 5 000 € pour les travailleurs occasionnels. Vous pouvez accéder à cette vue en sélectionnant l'option "Résumé Annuel" dans le menu latéral.

Période du Rapport

Le rapport annuel est calculé "sur une base de caisse," couvrant la période de décembre de l’année précédente à novembre de l’année sélectionnée. Cela est dû au fait que les paiements sont effectués le mois suivant (par exemple, pour l’année 2025, le rapport inclut les données de décembre 2024 à novembre 2025).

Détails de l'Écran

L’écran comprend les éléments principaux suivants :

  • Menu Déroulant: En haut, un menu déroulant vous permet de sélectionner l’année souhaitée (par exemple, 2024, 2025, etc.).
  • Tableau de Données: Un tableau résume vos gains avec les colonnes et lignes suivantes :
    • Colonnes: Catégorie, Brut, Net, TVA.
    • Lignes: Commandes, Ajustements, Pourboires, Total.
  • Informations sur les Livraisons: Sous le tableau, il est indiqué :
    • Le nombre de livraisons effectuées pendant l’année.
    • Le nombre de livraisons manquantes pour atteindre le bonus (les riders italiens reçoivent un bonus tous les 2 000 livraisons).

Comment Utiliser le Rapport Annuel

  • Assurez-vous d’avoir saisi les commandes et importé les relevés pertinents (voir Saisie Manuelle, Saisie par OCR et Importation de Relevé).
  • Sélectionnez l’année souhaitée dans le menu déroulant pour afficher les données correspondantes.
  • Analysez les valeurs dans le tableau pour comprendre vos gains bruts, nets et la TVA associée pour chaque catégorie (commandes, ajustements, pourboires) et le total global.
  • Vérifiez le nombre de livraisons effectuées et celles manquantes pour le bonus afin de planifier votre travail et atteindre l’objectif des 2 000 livraisons.
