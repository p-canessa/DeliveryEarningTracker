Paramètres

L'écran des paramètres vous permet de personnaliser l'application selon vos préférences. Vous pouvez accéder à cette vue en sélectionnant l'option "Paramètres" dans le menu latéral.

Options Disponibles

L'écran des paramètres inclut les options suivantes :

  • Symbole de la Monnaie: Vous permet de sélectionner la devise à utiliser dans l'application (par exemple, €, $, £). La valeur par défaut est €.
  • Mode Nuit: Un interrupteur pour activer ou désactiver le thème sombre de l'application. S'il est désactivé (valeur par défaut), l'application utilise le thème clair.
  • Sauvegarde des Données: Crée une copie de la base de données de l'application dans un dossier de votre choix sur votre appareil. Utile pour sauvegarder vos données en cas de besoin.
  • Restaurer les Données: Vous permet de restaurer la base de données de l'application à partir d'une sauvegarde précédemment créée. Assurez-vous d'avoir un fichier de sauvegarde valide.
  • Réinitialiser les Paramètres: Réinitialise tous les paramètres de l'application à leurs valeurs par défaut. Note : cette action n'affecte pas les données enregistrées (par exemple, les commandes ou relevés).
  • Supprimer les Publicités: Vous permet de vous abonner pour supprimer les publicités et débloquer toutes les fonctionnalités de l'application.
  • Partager des Statistiques Anonymes: Un interrupteur pour autoriser l'envoi de données agrégées anonymisées. En activant cette option (désactivée par défaut), vous aidez les développeurs à améliorer l'application.

Comment Utiliser les Paramètres

  • Accédez à l'écran des paramètres depuis le menu latéral.
  • Modifiez les options selon vos préférences :
    • Sélectionnez une devise différente si vous n'utilisez pas l'euro.
    • Activez le "Mode Nuit" pour une expérience visuelle plus confortable dans des conditions de faible luminosité.
    • Effectuez une sauvegarde des données régulièrement pour éviter de perdre des informations importantes.
    • Utilisez "Restaurer les Données" uniquement si vous avez une sauvegarde valide et souhaitez écraser les données actuelles.
  • Si vous rencontrez des problèmes avec les paramètres, vous pouvez utiliser l'option "Réinitialiser les Paramètres" pour revenir aux valeurs par défaut.
