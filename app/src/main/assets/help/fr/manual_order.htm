Saisie Manuelle

La fonction de saisie manuelle vous permet d'ajouter une commande directement depuis l'écran principal, en saisissant des détails tels que le nombre de livraisons, le total gagné et le temps passé.

Comment Insérer une Commande Manuellement

  1. Depuis l'écran principal, touchez le bouton "NOUVELLE COMMANDE".
  2. Saisissez les détails de la commande :
    • Date: La date à laquelle la commande a été effectuée, automatiquement définie à la date d’aujourd’hui.
    • Paiement de Base: La valeur de la ligne "Commandes Livrées" dans l’application Rider de Deliveroo.
    • Espèces Collectées: La valeur de la ligne correspondante, si présente.
    • Paiement Supplémentaire: La valeur du paiement supplémentaire, si présent.
    • Pourboire: La valeur du pourboire reçu via l’application, si présent.
    • Pourboire en Espèces: La valeur du pourboire reçu en espèces.
    • Nombre de Commandes: Le nombre de commandes livrées dans le même lot.
    • Temps Passé: Le temps pris pour livrer les commandes, pouvant être exprimé en minutes ou sous forme d’intervalle en heures et minutes comme dans l’application Rider (hh:mm-hh:mm).
  3. Confirmez la saisie pour enregistrer la commande.

Comment Modifier une Commande

Pour modifier une commande existante, suivez ces étapes :

  1. Depuis l'écran principal, repérez la commande que vous souhaitez modifier.
  2. Touchez la commande pour ouvrir l’écran de modification (le même écran utilisé pour la saisie manuelle).
  3. Mettez à jour les détails, tels que le nombre de livraisons, le paiement de base, le temps passé ou les pourboires.
  4. Confirmez les modifications pour enregistrer la commande mise à jour.

Si vous préférez insérer une commande en scannant une capture d’écran, consultez la section Saisie par OCR. Pour supprimer une commande, voir la section Supprimer une Commande.
