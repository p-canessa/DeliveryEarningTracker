Réconciliation

Cet écran vous permet de comparer les données des commandes quotidiennes extraites d’un relevé de paiement avec celles saisies manuellement ou via OCR. Vous pouvez accéder à cette vue en sélectionnant l’option "Réconciliation" dans le menu latéral.

Détails de l'Écran

L’écran comprend les éléments principaux suivants :

  • Navigation entre les Mois: Utilisez les deux boutons et pour naviguer entre les mois et afficher les données de réconciliation correspondantes.
  • Tableau de Comparaison: Le tableau affiche les données des commandes quotidiennes (nombre et montant) extraites du relevé, à côté de celles saisies manuellement ou via OCR.
    • Si les valeurs diffèrent, le texte dans le tableau est affiché en rouge.
    • En cliquant sur une ligne avec des valeurs différentes, vous accédez à la liste des commandes de ce jour (voir Visualisation des Commandes).
  • Espèces Collectées et Pourboires: Sous le tableau des commandes, deux lignes montrent les totaux des "Espèces Collectées" et des "Pourboires" :
    • Si les valeurs diffèrent de celles du relevé, le texte est affiché en rouge.
    • En cliquant sur une ligne différente, vous accédez à la liste des commandes avec des espèces collectées et/ou des pourboires différents de 0.

Comment Utiliser la Réconciliation

  • Assurez-vous d’avoir importé un ou plusieurs relevés via la fonction Importation de Relevé.
  • Saisissez les commandes manuellement ou via OCR (voir Saisie Manuelle et Saisie par OCR).
  • Sélectionnez le mois souhaité à l’aide des boutons de navigation pour afficher les données de réconciliation.
  • Vérifiez le tableau pour identifier les éventuelles divergences (en rouge) et cliquez sur les lignes pour voir les détails des commandes et corriger les erreurs.
