Importation de Relevé

La fonction d'importation de relevé vous permet de lire les données du relevé de fin de mois envoyé par Deliveroo. Ces données peuvent être utilisées pour générer des résumés trimestriels et annuels, ainsi que pour effectuer la réconciliation des commandes saisies.

Comment Importer un Relevé

  1. Sélectionnez la fonction "Importer un Relevé" dans le menu latéral.
  2. Sélectionnez un document nommé `invoice_*.pdf` depuis votre appareil via le bouton "IMPORTER RELEVÉ" (cela peut nécessiter de visionner une publicité).
  3. L'application extraira automatiquement les données du fichier PDF. Vous pouvez importer plusieurs relevés pour le même mois, à condition qu'ils aient des numérotations différentes.
  4. Si le relevé n'est pas déjà dans la base de données, il sera ajouté et un message de succès sera affiché. Sinon, un message d'erreur apparaîtra.
  5. À ce stade, le relevé peut être consulté dans la section Afficher le Relevé Mensuel.
