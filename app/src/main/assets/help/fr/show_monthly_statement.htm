Affichage du Relevé

Cet écran vous permet de visualiser les données d’un relevé précédemment importé. Vous pouvez accéder à cette vue en sélectionnant l’option "Afficher le Relevé Mensuel" dans le menu latéral.

Détails de l'Écran

L’écran affiche les détails du relevé pour le mois sélectionné, tels que les gains totaux, les pourboires et autres informations pertinentes. Utilisez les deux boutons et pour naviguer entre les relevés présents dans la base de données.

Comment Accéder aux Relevés

  • Assurez-vous d’avoir importé un ou plusieurs relevés via la fonction Importation de Relevé.
  • Sélectionnez le relevé souhaité pour afficher les données correspondantes.
